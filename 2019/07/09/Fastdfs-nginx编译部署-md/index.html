Fastdfs+nginx编译部署.md

发表于 2019-07-09

系统环境

  • 华为泰山2280服务器
  • 中标麒麟操作系统

安装包下载

  • libevent-2.1.8-stable.tar.gz
  • libfastcommonV1.0.39
  • fastdfs5.08
  • fastdfs-nginx-module_1.20.tar.gz

安装步骤：

1. libevent-2.1.8-stable.tar.gz

2. libfastcommonV1.0.39

3. fastdfs5.12 – tracker服务

4. fastdfs5.08 – storage服务

5. 测试

6. 安装fastdfs-nginx-module_1.20.tar.gz

首先需要安装nginx nginx 1.15.1

添加nginx为系统服务

cat /usr/lib/systemd/system/nginx.service

修改fastdfs-nginx-module配置文件

对nginx重新config

修改nginx配置文件 重启nginx

将libfdfsclient.so拷贝至/usr/lib下

cp /usr/lib64/libfdfsclient.so /usr/lib/

启动nginx

systemctl start nginx
到此处，即可直接从浏览器访问上传的文件（图片）
